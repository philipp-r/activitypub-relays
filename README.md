# activitypub-relays

If you want to add a relay to Mastodon or Pleroma:

#### Mastodon instance
- Navigate to /admin/relays
- Click "Add New Relay"
- Paste one of the urls below *as is*
- Click "Save and Enable"

#### Pleroma Instance
- Copy the URL & replace */inbox* with */actor*
